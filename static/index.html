Comment connecter GA4 en temps réel

Étape 1 : Aller sur console.cloud.google.com → Créer un projet ou en sélectionner un
Étape 2 : Activer l'API Google Analytics Data API (GA4)
Étape 3 : Créer un Service Account avec accès "Viewer" sur votre propriété GA4
Étape 4 : Télécharger le fichier JSON de credentials
Étape 5 : Ajouter le Service Account email comme "Viewer" dans GA4 → Admin → Property Access Management
Étape 6 : Ajouter les variables d'environnement : GA4_PROPERTY_ID et GA4_CREDENTIALS_JSON sur Railway
